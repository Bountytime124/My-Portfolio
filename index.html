My Project

Automatic gel press machine

This event is part of the competition in the Inter-class Science Project Work Competition 2022, a science project competition within Wat Sing High School. which my friend and I chose to do a project called automatic gel press machine.
During Covid, this project, the three of us, came up with the idea that if there was a sensor machine that automatically produced gels, it might reduce the number of people infected with Covid. The working principle of this sensor is to detect a hand within 7-10 cm. If it is within that distance, it will work immediately. We have written code in C++ (Arduino nano) to connect with the alcohol gel to make it work.
